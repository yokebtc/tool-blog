# tool-blog 博客系统

此博客来源于 [Tale](https://github.com/otale/tale)，由于博主长时间不更新，在基础上修改一些页面布局、文章背景图片、页面静态化等特性，打造属于自己的博客平台

演示站点：http://blog.danglaoshi.app

[开始使用](https://github.com/yokebtc/tool-blog/wiki)&nbsp; | &nbsp;[视频教程](https://github.com/yokebtc/tool-blog/wiki/07.-%E8%A7%86%E9%A2%91%E6%95%99%E7%A8%8B)&nbsp;

## 特性

+ 设计简洁，界面美观
+ Markdown 文章发布
+ 自定义文章链接
+ 支持多主题
+ 支持插件扩展
+ 支持 Emoji 表情
+ 支持网易云音乐播放
+ 支持附件和数据库备份
+ 部署简单，不依赖 Tomcat
+ 无需数据库，内嵌 Sqlite
